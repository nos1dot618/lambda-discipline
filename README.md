## Getting Started

```bash
mkdir build
cd build
cmake ..
make
```

## Usage

```bash
usage: lbd [options]

options:
-f, --file <filename>   Specify input source file to run
-h, --help              Show this help message and exit
-d, --debug             Enable debug mode
-r, --repl              Run in interactive REPL node
```
